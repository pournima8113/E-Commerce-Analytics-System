# 🛒 E-Commerce Analytics System (SQL Project)

## 📌 Project Overview
This project focuses on analyzing e-commerce data using MySQL. It helps in understanding customer behavior, product performance, and overall sales trends through structured SQL queries. The goal is to provide actionable business insights for an e-commerce platform.

## 🎯 Objectives
- Analyze customer purchasing behavior and spending patterns.
- Identify top-selling products and categories.
- Track monthly revenue trends and growth.
- Generate business insights using SQL joins and aggregations. 

## 🛠️ Tools & Technologies
Dtabase: MySQL  
Language: SQL (Structured Query Language)

## 🗄️ Database Schema
The project consists of the following key tables:

- Customers: Stores details like Customer ID, Name, and Email.
- Products: Contains product names, categories, and prices.
- Orders: Tracks order dates, amounts, and customer links.
- Order_Items: Detailed breakdown of products within each order.  

## 🚀SQL Features Used

- Joins: (Inner Join, Left Join for connecting customer and order tables)
- Aggregations: (SUM, AVG, COUNT to calculate revenue and order frequency)
- Grouping & Filtering: (GROUP BY, HAVING, WHERE)
- Ordering: (To rank products and customers)

## ⚙️How to Run the Project
- Clone this repository.
- Open MySQL Workbench or any SQL editor.
- Run schema.sql to create tables.
- Run data_insertion.sql to load sample data.
- Execute queries from queries.sql to see results.
